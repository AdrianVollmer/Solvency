# Code Review

Perform a code review. Review the parts:

- Rust
- JavaScript
- HTML Templates
- CSS

The focus should be on these points in this order:

- Security
- Gross performance issues
- Potential performance improvements
- Readability and maintainability (clean code, DRY, little code overlap,
  avoid code duplication, no exzessive function length or argument
  count)
- Tests

Create issues for the top 10 findings
